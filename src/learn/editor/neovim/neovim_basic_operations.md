# Neovim的基本操作

## 打开与关闭 Neovim

- 打开 Neovim：在终端中输入 `nvim`，然后按回车键。
- 关闭 Neovim：在命令模式下输入 `:q`，然后按回车键。如果有未保存的更改，使用 `:q!` 强制关闭。
- 保存并关闭：在命令模式下输入 `:wq`，然后按回车键。

## 光标移动操作

### 简单移动

- 基础移动

| 按键 |   说明   |
|:--:|:------:|
| j  | 向下移动光标 |
| k  | 向上移动光标 |
| h  | 向左移动光标 |
| l  | 向右移动光标 |

- 单词移动

| 按键 |         说明          |
|:--:|:-------------------:|
| w  |     移动到下一个单词的开头     |
| W  | 移动到下一个单词的开头（忽略标点符号） |
| b  |     移动到上一个单词的开头     |
| B  | 移动到上一个单词的开头（忽略标点符号） |
| e  |     移动到下一个单词的结尾     |
| E  | 移动到下一个单词的结尾（忽略标点符号） |

- 行内移动

| 按键 |  说明   |
|:--:|:-----:|
| 0  | 移动到行首 |
| $  | 移动到行尾 |

- 屏幕移动

| 按键 |     说明      |
|:--:|:-----------:|
| H  |   移动到屏幕顶部   |
| M  |   移动到屏幕中部   |
| L  |   移动到屏幕底部   |
| gg |   移动到文件开头   |
| G  |   移动到文件结尾   |[
| zz | 将当前行移动到屏幕中部 |
| zt | 将当前行移动到屏幕顶部 |
| zb | 将当前行移动到屏幕底部 |

### 组合移动

|    按键    |   说明   |
|:--------:|:------:|
| Ctrl + f |  向下翻页  |
| Ctrl + b |  向上翻页  |
| Ctrl + d | 向下滚动半页 |
| Ctrl + u | 向上滚动半页 |

### 高阶移动

| 按键 |        说明        |
|:--:|:----------------:|
| 2j |      向下移动两行      |
| 3k |      向上移动三行      |
| 4w | 移动到下一个单词的开头，重复四次 |
| 5b | 移动到上一个单词的开头，重复五次 |
| 6e | 移动到下一个单词的结尾，重复六次 |
